### px,em与rem

```html
<html>
    <head>
        <style>
            html:{
                font-size:20px
            }
            div:{
                font-size:30px;
            }
            h1:{
                font-size:25px;
                margin-top:2em;
            }
            h2:{
                font-size:2rem;
            }
        </style>
    </head>
    <body>
        <div>
            <h1> </h1>
            <h2> </h2>
        </div>
    </body>
</html>
```

说出h1的margin-top为多少px，h2和font-size为多少px
